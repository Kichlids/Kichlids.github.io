Sooner Competitive Robotics

2019-2021 IGVC

Software team member of the Intelligent Ground Vehicle Competition. Developed software that obtains robot's location using a GPS and processes camera images to be used for navigation using Robot Operating System and XBox Kinect.

Game Dev Association

Tower Defense

A variation of tower defense genre developed from scratch in Unity Engine. A unique feature of this game is that the player can create the path and the enemies can destroy them.

ML Blackjack

Implementation of machine learning in Blackjack. Player plays with a machine learning agent trained with the Q-Learning algorithm against the dealer. Project originally developed for CS 4033 Machine Learning and implemented for Unity Engine with the added playability aspect for the player.

Ecosystem Simulator

Simple ecosystem simulator developed on the Unity Engine consisting of two different species programmed to search for food, water, and reproduce as well as flee from predators on a procedurally generated environment.
